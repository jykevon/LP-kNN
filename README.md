# Q2 ML Project: LP-kNN

Supervised learning algorithms often rely on large and densely labeled datasets, which are not always available in real-world settings. One such scenario occurs during the early stages of presidential elections, when only partial results are released while accurate predictions and inferences are still desired. In these settings, traditional supervised learning algorithms can suffer from high variance and unreliable generalization due to limited labeled data.

In this paper, we propose \emph{Lattice-Propagated $k$-Nearest Neighbors (LP-$k$NN)}, a hybrid classification algorithm that extends the neighbor-based intuition of $k$-Nearest Neighbors by propagating labels across a discretized lattice. By framing classification as an iterative lattice-filling process, LP-$k$NN is able to infer labels in regions of the feature space where training data is sparse or unavailable. This approach enables effective classification under limited data availability and provides a structured alternative to standard instance-based methods.
